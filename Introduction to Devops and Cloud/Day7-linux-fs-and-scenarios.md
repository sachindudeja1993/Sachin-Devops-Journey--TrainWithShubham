- /etc (Configuration Files) - Stores system and service configuration files.
- /var/log (Log Files) - Stores system and application logs.
- /tmp (Temporary Files) - Stores temporary files.
- /bin (Essential Commands) - Contains basic Linux commands.
- /usr/bin (User Commands) - Stores most user-level programs.
- /opt (Optional Software) - Stores third-party applications.
